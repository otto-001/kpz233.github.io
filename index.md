# 核心机制：为何它能重塑您的交易纪律？

### 1. 独创积分闯关系统
将枯燥的纪律化为闯关的乐趣。每一次盈利，都是向着更高积分迈进；每一次亏损，都是对策略的警示。我们用最直观的分数，量化您的交易表现。
*   **盈利加分：** 每完成一轮累计盈利目标，积分 `+100`。
*   **亏损扣分：** 任何一笔止损，积分 `-200`。严苛的惩罚机制，只为让您对每一次亏损都心存敬畏。
*   **等级挑战：** 随着积分提升，EA的止盈止损规则会变得更加严格，模拟真实的资金挑战赛，不断锤炼您的交易心态。

### 2. 动态目标止盈止损
告别固定点数的机械式止盈。本EA的止盈位（TP）是根据您当前“**本轮累计盈利目标**”动态计算的。这意味着，每一笔订单的终点，都是为了服务于您本轮的总体盈利计划，而非短视的几十个点。止损位（SL）则与之对应，确保风险回报比的合理性。

### 3. 自动化风控模块
开仓后，您只需专注市场。风控，交给EA。
*   **智能保本功能 (Breakeven):** 当浮盈达到预设目标的一定比例时，EA会自动将您的止损位移动到“最低盈利”位置，让已经到手的利润不再溜走，让利润自由奔跑。
*   **安全垫机制 (Safety Net):** 我们已优化底层逻辑，当您设置的盈利目标过小时，EA会自动应用一个“最低盈利安全垫”，确保止盈时不会因为滑点和成本而意外亏损。

### 4. 专注的交易界面
*   **自动隐藏面板：** 手动开仓后，MT4/MT5自带的“一键交易”面板会自动隐藏，为您提供一个干净、无干扰的看盘界面。
*   **开仓即沉浸，平仓即复盘：** 杜绝您在持仓过程中的手动干预冲动，让您真正做到“只负责开仓，让规则飞一会儿”。

# 常见问题 (FAQ)

### Q1: 这个EA是全自动交易机器人吗？
**A:** 不是。这是一个**手动交易辅助工具 (Trade Manager)**。您需要自己判断市场并手动开仓，EA只负责开仓后的所有管理工作，包括设置并执行SL/TP、保本等。

### Q2: EA支持哪些交易品种和时间周期？
**A:** 理论上支持所有品种和周期。因为EA的管理逻辑是基于您开仓后的价格和盈利目标，与具体品种和周期无关。我们推荐在波动性与点差较为均衡的主流品种上使用，如XAUUSD, EURUSD等。

### Q3: MT4和MT5版本有什么区别？
**A:** 核心功能和逻辑完全一致。您可以根据您习惯使用的平台来选择版本。

### Q4: 如何安装和激活？
**A:** 联系我们获取EA文件后，我们会提供一份详细的图文安装教程。您只需将EA文件放入正确的文件夹，加载到图表上，然后在参数中输入我们为您生成的唯一激活码即可。
